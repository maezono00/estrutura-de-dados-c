ÁRVORE

FUNDAMENTO
ÁRVORE SÃO ESTRUTURA QUE PERMITE ORGANIZAR DADOS ORGANIZAR DADOS DE FORMA HIERÁRQUICA. ESTA ESTRUTURA É UM TIPO ESPECIAL DE ÁRVORE, DENOMINADA COMO ÁRVORE BINÁRIA QUE PERMITE IMPLEMENTAR A OPERAÇÃO DE BUSCA DE FORMA EFICIENTE.

UMA ÁRVORE A É UMA ESTRUTURA HIERÁRQUICA COMPOSTA POR N >= 0 NÓS. SE N = 0, DIZEMOS QUE A ÁRVORE A É VAZIA, CASO CONTRÁRIO;
	*EXISTE UM NÓ ESPECIAL EM A DEMONIMADO COMO RAIZ;
	
OS DEMAIS NÓS DE A SÃO ORGANIZADOS EM A1, AZ... AN ESTRUTURAS DISJUNTAS DENOMINADAS SUBÁRVORE DE A.

EXEMPLO:


PODEMOS REFERENCIAR ÁRVORE COM N NODES, OU SEJA, "VÉRTICES" ANALOGICAMENE REFERENCIAMOS ÁRVORE COMO ÁRVORE DE DIRETÓRIO

PARA QUE POSSAMOS MAPEAR UMA ÁRVORE ADOTA-SE O SEGUINTE:

*NÍVEL DO NODO;
*GRAU DO NODO;
*ALTURA DA ÁRVORE;
